# لاگ ان فارم پروجیکٹ

## تعارف
یہ ایک سادہ رجسٹریشن فارم ہے جو HTML، CSS اور JavaScript سے بنایا گیا ہے۔

## فائلیں
- **index.html** — فارم کا ڈھانچہ
- **style.css** — فارم کی ڈیزائن
- **script.js** — پاس ورڈ چیک کرنے کا کوڈ

## پاس ورڈ کی شرائط
- کم از کم 8 حروف
- کم از کم ایک بڑا حرف
- کم از کم ایک چھوٹا حرف
- کم از کم ایک نمبر
- کم از کم ایک خاص علامت

## چلانے کا طریقہ
index.html فائل کو براہ راست براؤزر میں کھولیں۔
